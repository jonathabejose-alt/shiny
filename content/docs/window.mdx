# window

```lua
local Window = WindUI:CreateWindow({ 
  Title="My Hub", 
  Icon="door-open", 
  Author="by .ftgs", 
  Folder="MyHub", 
  Size=UDim2.fromOffset(580,460), 
  MinSize=Vector2.new(560,350), 
  MaxSize=Vector2.new(850,560), 
  ToggleKey=Enum.KeyCode.LeftShift, 
  Transparent=true, 
  Theme="Dark", 
  Resizable=true, 
  SideBarWidth=200, 
  BackgroundImageTransparency=0.42, 
  HideSearchBar=true, 
  ScrollBarEnabled=false 
})
```

Actions: Window:Open() / Window:Close() / Window:Toggle()

Elements: Window:LockAllElements() / Window:UnlockAllElements() / Window:GetAllElements()

Triggers: Window:OnClose(fn) / Window:OnOpen(fn) / Window:OnDestroy(fn)
