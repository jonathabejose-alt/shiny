## Frequently Asked Questions

**What is WindUI?**
WindUI is a modern UI library for Roblox Script Hubs and Luau environments, designed to be easy to use and visually appealing.

**What is WindUI Shiny?**
WindUI Shiny is a refined fork of the original WindUI, with visual improvements, bug fixes, and extra elements while maintaining full API compatibility.

**How do I lock elements?**

```lua
element:Lock()   -- lock
element:Unlock() -- unlock
```

**How do I use custom icons?**
WindUI uses [Lucide icons](https://lucide.dev). Pass the icon name as a string:

```lua
Tab:Button({ Title = "My Button", Icon = "bird" })
```

To use themed coloring on custom icons, set `IconThemed = true`.

**How do I disable the OpenButton?**
Pass `OpenButton = false` inside `CreateWindow()`.
