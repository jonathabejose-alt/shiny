# Installation

```bash
git clone https://github.com/Footagesus/WindUI.git
cd WindUI && npm install && aftman install
```

**Commands:**

| Command | Description |
|---------|-------------|
| `npm run dev` | Development server |
| `npm run build` | Production build |
| `npm run live-build` | Live build server (localhost:8642) |
